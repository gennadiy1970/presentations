HTML Forms

Формы: хранение данных

Один из популярных видов хранения данных это запись ключ= значение

Ключи в пределах одного документа должны быть уникальны.

Пример: пара "атрибут тега" <img src="logo.jpg" >
ключ - src, значение logo.jpg

Формы: задаем ключ и задаем значение

Пару ключ= значение можно использовать чтобы задать произвольный ключ: key="one" и произвольное значение value="some"

Именно так в <input> задается ключ - с помощью атрибута name и значение - с помощью атрибута value. При этом value может перезаписаться пользовательским значением.

Пример: пара name: value; <input type="range" name="volume" value="0.5" min="0" max="1" step="0.1">
ключ - volume, значение 0.5 Посмотреть работающий пример

Формы: генерация пар "ключ-значение"

<form action="#"> // "#" - адрес, куда будут направлены данные
  <input type="text" name="login" placeholder="login">
  <input type="text" name="pass" placeholder="pass">
  <input type="submit">
</form>

Если ввести логин "cool" и pass "123", то в результате в строке браузера будет подобный текст:

keyvalue.html?login=cool&pass=123#
Посмотреть работающий пример

method GET vs POST - скрываем данные

<form action="#" method="post">
  <input type="text" name="login" placeholder="login">
  <input type="text" name="pass" placeholder="pass">
  <input type="submit">
</form>

Метод POST выносит данные из заголовка сообщения. В результате в строке браузера будет подобный текст:

keyvalue.html# // в строке остался только адрес из action
